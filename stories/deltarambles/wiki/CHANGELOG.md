# The Delta Archive V3: 'Changelog'

## Format

`- (author), ( dd month-name YYYY | 24-hour time when finished edit) | (file/s) changed the root is usually the wiki/. | (brief summary of what changed)`

`- jmb, 25 May 2026 | 22:05 | characters/mijo.html | template and stub entry`

Add `/\*` to the end of your folder if you have changed or added the folder in a significant way (on discretion) 

## Explanation

Entries are separated by month, and if there are enough changes, old changes will be moved to the changelog archive per... year? (or when it gets too long)
